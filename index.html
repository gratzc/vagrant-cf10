ColdFusion 10 Vagrant Environment
And Cloudy With A Chance Of Tests

This project is to demostrate how to setup a testing enviroment using Cloudy With A Chance Of Tests. This includes syntax, qpscanner, copy-paste detections, and varscoper inspections for ColdFusion. Mxunit is used for an example of unit and acceptance testing. We also include jslint and csslint inspections.

To get started please read the README.md. Also review the wiki.

Quick Setup

  
    $ cd vagrant-cf10
    $ librarian-chef install
    $ vagrant up
